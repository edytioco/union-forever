README

Because this bot has a couple of dependencies, so if you're running Windows, here are a few commands you need to install them:
npm install --save discord.js dotenv
npm install --save ytdl-core
npm install --save @discordjs/opus
npm install ffmpeg-static --save




The primary sources I used to learn how to build this bot are the following:
[1] Discord bot tutorial by Renemari Padillo
https://medium.com/davao-js/2019-tutorial-creating-your-first-simple-discord-bot-47fc836a170b

[2] Discord.js guide
https://discordjs.guide/popular-topics/miscellaneous-examples.html#play-music-from-youtube

[3] Discord music bot tutorial
